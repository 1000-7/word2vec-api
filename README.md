word2vec-api
============

Simple web service wrapping a Word2Vec as implemented in Gensim.
Example call: 
 curl http://127.0.0.1:5000/wor2vec/n_similarity/ws1=Sushi&ws1=Shop&ws2=Japanese&ws2=Restaurant
